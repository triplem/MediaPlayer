About MediaPlayer

A java project that uses the OpenHome API (http://www.openhome.org/wiki/Oh:Overview) to implement a UPNP Renderer.

Project Modules

This project has declared the following modules:

Name Description
MediaPlayer Core Core Module for the OpenHome MediaPlayer
MediaPlayer Plugin AlarmClock Plugin which allows you to create an alarm clock
MediaPlayer Plugin CustomButton A java project that uses the OpenHome API (http://www.openhome.org/wiki/Oh:Overview) to implement a UPNP Renderer.
